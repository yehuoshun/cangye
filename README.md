# 藏叶 🍃

**个人收藏聚合器** — Windows 桌面端，手动创建虚拟文件夹聚合多源文件，统一浏览与预览。

> 藏 = 收藏收纳，叶 = 文件如叶

## 功能

- 📁 **虚拟文件夹** — 创建虚拟文件夹聚合来自不同来源的文件
- 🔗 **多源路径** — 绑定本路径、网页链接到同一个文件夹（支持 `115:\`、`tg:\`、`notion:\` 等前缀）
- 📂 **聚合浏览** — 网格/列表视图，统一浏览所有来源的文件
- 👁️ **内嵌预览** — 图片、视频、音频、文本、Markdown、代码预览
- 🏷️ **标签系统** — Notion 风格的彩色标签，动态搜索、快速筛选
- 📊 **总览仪表盘** — 按类型统计文件数量
- 🌗 **暗色/亮色主题** — 自由切换
- 🖥️ **三种布局** — 侧边栏/顶部导航/双栏对开

## 快速开始

### 下载

从 [Releases](https://github.com/yehuoshun/cangye/releases) 下载最新版本，双击运行即可自动打开浏览器。

### 从源码构建

```bash
# 1. 构建前端
cd frontend
npm install
npm run build:skip-lint

# 2. 构建后端
cd ../backend
go build -o ../cangye.exe .

# 3. 运行
./cangye.exe
```

访问 `http://127.0.0.1:27138` 即可使用。

## 使用说明

### 创建虚拟文件夹

1. 🔄 点击「刷新」按钮加载根目录
2. ➕ 点击「+ 新建文件夹」
3. 填写名称和图标
4. 创建后点击文件夹进入

### 绑定路径

进入文件夹后：

1. ➕ 点击「添加路径」
2. 输入带前缀的路径（例如 `115:\学习\Java\`、`tg:\Java频道`）
3. （可选）开启自动扫描
4. 点击「扫描」同步文件

### 添加虚拟文件

不需要扫描的文件可以手动添加：

1. ➕ 点击「添加文件」
2. 输入路径（支持前缀）
3. 设置显示名称（可选）

### 标签系统

- 文件和文件夹都可绑定标签
- 标签输入支持动态搜索（debounce 300ms）
- 无匹配时回车直接创建新标签
- 10 种颜色预设

## 技术架构

```
浏览器（Vue 3） ←→ HTTP REST API (Go localhost:27138) ←→ SQLite
                                                     ←→ 本地文件 I/O
                                                     ←→ exec 调系统程序
```

- **后端**: Go + gorilla/mux + SQLite
- **前端**: Vue 3 + TypeScript + Vite
- **前端嵌入**: embed.FS 内嵌到 Go 二进制
- **单文件分发**: 双击 exe → 起服务 → 自动开浏览器

## 前缀系统

| 前缀 | 类型 | 说明 |
|------|------|------|
| `115:\` | local | CD2 挂载的本地路径 |
| `tg:\` | web | Telegram 网页链接 |
| `notion:\` | web | Notion 网页链接 |

可在设置页面自定义前缀映射。
